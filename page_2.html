UMSI Peer programming tutoring

We have UMSI programming peer tutors available for both SI and SIADS courses!

We offer peer programming help for the following courses, in addition to general python support:
  1. SI 106 Programs, Information, and People
  2. SI 206 Data-Oriented Programming
  3. SI 506 Programming I
  4. SI 507 Intermediate Programming
We also offer tutoring and programming support in MADS for the following course:
  1. SIADS 505 Data Manipulation
If you have any questions, please contact us at umsi.academicsuccess@umich.edu for more details!
